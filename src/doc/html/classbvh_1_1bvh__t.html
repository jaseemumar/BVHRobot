Detailed Description

The bvh data structure.

The bvh data structure has primarily two components one to store the skeleton hierarchy, another to store the motion data. A bvh data structure may not even have any motion data. It would be desirable to extend this class so that it can handle multiple skeleton hierarchies from one file but since the exact specification of this is not known it is left for the future.
